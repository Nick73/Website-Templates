About This Template

Pixel Arcade is a retro-styled HTML/CSS/JS starter with neon glow, pixel icons, scanlines, a CRT sheen, and a starfield backdrop. Built for that late-90s/early-2000s arcade feel.

  • Pixel font via Google Fonts (Press Start 2P).
  • Pure CSS neon borders + flicker + marquee.
  • Canvas starfield (tiny, no deps).
